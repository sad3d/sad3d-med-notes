Написание заключения ЭЭГ

Структура заключения

  1. Шапка (бланк учреждения)
  2. Информация о пациенте
  3. Техническая информация об исследовании
  4. Описание
    • фоновая запись
      • нормальные паттерны
      • патологические находки (в случае наличия)
    • активирующие пробы
    • сон
      • нормальные паттерны
      • патологические находки (в случае наличия)
  5. Заключение
  6. Клинический комментарий (опционально)

Информация о пациенте

  1. Дата проведения исследования
  2. Фамилия, имя, отчество
  3. Дата рождения и возраст
  4. Повод обращения / краткий анамнез
  5. Цель исследования (исследование ЭЭГ в динамике / исключение эпилепсии / контроль эффективности терапии / контроль после хирургического вмешательства / предхирургическое обследование / контроль после отмены терапии)
  6. Депривация сна (не проводилась / полная / частичная)
  7. Принимаемые препараты (противоприступные и психофармакологические)

Техническая информация об исследовании

  1. Тип ЭЭГ исследования (стандартное, дневной/ночной/суточный (видео-) ЭЭГ мониторинг)
  2. При необходимости можно указывать место проведения исследования (амбулаторный / в палате ОАРИТ)
  3. Система расположения электродов (международная схема “10-20” / “10-10”)
  4. Тип ЭЭГ электродов (скальповые / кортикографические / глубинные) 
  5. Дополнительные каналы (ЭКГ, ЭМГ с указанием локализации, ЭОГ, электроды скуловой дуги)
  6. Длительность исследования (30 минут / 1 час / 4 часа / 10 часов…)
  7. Протокол исследования (с функциональными пробами, в состоянии активного и пассивного бодрствования, а также дневного/ночного сна)

Описание

  1. Описание доминирующего затылочного ритма (в случае, если он зарегистрирован. Если нет — «доминирующий затылочный ритм отчетливо не зарегистрирован»)

    1. регулярность (регулярный / фрагментарный)
    2. устойчивость (устойчивый / не устойчивый по частоте)
    3. модулированность (модулированный / слабо модулированный / немодулированный)
    4. частота
    5. амплитуда (максимальная)
    6. локализация (в затылочно-задневисочных отделах полушарий / в затылочно-задневисочно-теменных отделах полушарий и т.д.)
    7. асимметричность (без значимой разницы сторон / выраженно асимметричный)
    8. зональные различия (выражены отчетливо / удовлетворительно / сглажены)

  2. Другие физиологические паттерны бодрствования

    1. чаще всего мю-ритм («в лобно-центральных отделах с переменной латерализацией регистрировался аркообразный сенсомоторный ритм частотой около Гц, амплитудой до мкВ»)
    2. описание любого другого паттерна можно взять из глоссария / классификации Людерса / атласа / презентации

  3. Описание бета-активности (если была достаточно выраженной для описания)

    1. «В лобных отделах полушарий эпизодически регистрировались группы бета-волн частотой около 25 Гц, амплитудой до 15 мкВ»
    2. «В бодрствовании увеличенным индексом регистрировались группы бета-волн с амплитудным преобладанием в лобных отделах полушарий, частотой 20-26 Гц, амплитудой до 70 мкВ - паттерн «избыточная бета-активность».»

  4. Описание медленноволновой активности (в стандартном случае «медленноволновая активность регистрировалась диффузно, умеренным индексом, преимущественно волнами тета-диапазона, по амплитуде не превышает основной ритм»).

  5. Описание патологической активности в бодрствовании (замедление, эпилептиформная активность, периодические паттерны), включая её локализацию, морфологию, амплитуду, индекс

  6. Описание функциональных проб (см. лекцию по функциональным пробам).

    • проба с открыванием-закрыванием глаз: при проведении пробы патологические ЭЭГ феномены не регистрировались (если регистрировались - описание). Реакция активации сохранена.
    • проба с ритмичной фотостимуляцией проводилась на частотах 1, 2, 8, 10, 16, 18, 20, 25, 30, 50 Гц, с открытыми и закрытыми глазами, реакция усвоения ритма не зарегистрирована / зарегистрирована на частотах - Гц. Фотопароксизмальный ответ не зарегистрирован (если зарегистрирован - описать, классифицировать).
    • проба с гипервентиляцией: проводилась в течение 3 / 5 минут, в процессе выполнения отмечалось увеличение индекса медленноволновой активности, (в том числе в виде высокоамплитудных потенциалов с преобладанием в передних отделах до _ мкВ) с быстрым восстановлением фона после окончания пробы (физиологическая реакция) / отмечалась провокация патологической активности (указать какой именно) в виде увеличения её индекса / амплитуды.

  7. В случае регистрации сна - описание зарегистрированных нормальных паттернов сна, а также патологической активности (материалы последующих занятий)

Заключение

  1. Оценка фоновой ритмики бодрствования, физиологических паттернов сна и бодрствования, дифференцировки сна на стадии.

    • Корковая ритмика = фоновая ритмика + физиологические паттерны

  2. Описание патологических паттернов (в случае, если таковые зарегистрированы) - несколько более кратко, чем в описании, можно опустить амплитуду. Указание провоцирующих факторов для регистрации патологической активности, если таковые имелись (увеличение индекса в дремотном состоянии / после пробуждения и т.д.)

  3. Указание наличия или отсутствия патологических ЭЭГ феноменов при проведении активирующих проб

  4. Указание наличия или отсутствия эпилептиформной активности, эпилептических приступов и их электрографических паттернов.

  5. При наличии предыдущего исследования, сделанного в вашем учреждении - оценка динамики (без отрицательной динамики / без существенной динамики / положительная динамика в виде полной редукции эпилептиформной активности в бодрствовании и во сне и т.д.
